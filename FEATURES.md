# QuestyBook - AI 학습 코칭 시스템 기능 현황

> 최종 업데이트: 2026년 1월 6일

---

## 📖 프로젝트 개요

**QuestyBook**은 ACE Framework V5.2 기반의 AI 학습 코칭 시스템입니다.
노트북 스타일의 학습 플래너 UI와 멀티 에이전트 AI 코칭을 통해 개인화된 학습 경험을 제공합니다.

### 핵심 기술 스택
- **Frontend**: React + TypeScript + Zustand
- **Backend**: Hono.js + Bun Runtime
- **AI Engine**: questyCoachAgent (Multi-Agent System)
- **LLM**: Claude 4.5, Gemini 3, GPT-5 (OpenRouter)
- **Vector DB**: ChromaDB (768차원 임베딩)
